# **App Name**: BeautifulSoup&Food

## Core Features:

- Product Listing & Filtering: Display products in a grid with filters for category, price, and stock status.
- Custom Order Submission: Allow customers to submit requests for items not available in the main shop.
- Shopping Cart with Pickup/Delivery Options: Summarize selected items, allow quantity adjustments, and provide options for pickup or delivery.
- Admin Dashboard: Provide an interface for managing orders, users, inventory, and settings.
- Real-time Chat with Customers: Enable real-time communication between admins and customers. Floating chat icon
- Inventory Management: Enable admin to manage and monitor real-time stocks.
- Role and Permission management: Defining new roles for different part of the app.
- Homepage: A welcoming page with a hero section, about us section, testimonials, and featured products. Will have shop tab, soup tab, and custom order tab
- Shop Tab: A standard shopping tab for users to add items to cart, displaying products in a grid with filtering and pagination. All product sample should be different nigeria foodstuff
- Soup Page: A dedicated page that shows only soup products, allowing quantity selection, adding to cart/wishlist, and displaying/submitting customer reviews. All product samples should be different Nigeria soup
- Authentication Pages: Complete set of pages for user Sign Up, Login, Forgot Password, and Email Verification.
- Customer Account: Pages for users to view/update profile, order history, and notification preferences. wishlist
- Notification Bell: Bell icon for notification
- Sidebar Navigation: App has a sidebar for navigation
- PWA Support: Enable the application to be installable as a Progressive Web App (PWA) with the name BeautifulSoup&Food
- Pagination: Enable pagination for long lists of products or orders.
- User Management: Admin tab for managing users
- Setting: Admin tab for managing settings which have four tabs setting [homepage, footer, custom order setting for[ measure and service], roles [for setting roles and permission]
- Inventory: Admin tab for managing inventory
- Conversation: Admin tab for managing conversation
- Purchase Order: Admin tab for managing purchase order
- Accounting: Admin tab for managing accounting
- Analytics: Admin tab for managing analytics
- Profile Icon: Profile icon for all signed in user to see their order history, profile information, notification tab to allow them manage preferences, logout

## Style Guidelines:

- Primary color: Forest green (#4B9966) for a natural, fresh feel.
- Background color: Light mint (#F0FFF7) for a clean and calming environment.
- Accent color: Teal (#359283) for highlights and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif that is modern, friendly and readable.  Note: currently only Google Fonts are supported.
- Use consistent, simple icons from 'Shadcn UI' throughout the app to represent categories and actions.
- Maintain a clean, well-organized layout with clear visual hierarchy. Ensure app sidebars, pages, and forms are scrollable, work across all devices, and that all elements follow the principles of responsive design.
- Use subtle animations for loading states and transitions to provide a smooth and engaging user experience.
